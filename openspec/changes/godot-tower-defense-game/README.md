## 项目概述

一个基于 Godot 4.6 的 2D 塔防游戏，核心特色是**开放地图玩家自建路径**系统。

### 核心需求
- 3-5 种防御塔（基础炮塔、减速塔、范围塔、狙击塔、辅助塔）
- 4 种敌人类型（快速、坦克、飞行、再生）
- 开放地图，玩家自建路径
- A* 寻路系统，建塔时验证路径可达性
- 飞行单位遵循寻路但可飞过防御塔
- 初始金币 + 击杀奖励的资源系统
- 关卡制：通过 10 波进入下一关
- 使用占位符美术资源（ColorRect）

## 技术要点

### 寻路系统
- A* 算法实现
- 地面单位：不能穿过防御塔
- 飞行单位：可以飞过防御塔但仍遵循寻路
- 建塔时验证路径可达性（防止堵死）

### 地图设计
- 网格大小：20x15
- 单元格大小：32 像素
- 固定起点和终点
- TileMap 实现

### 资源管理
- 初始金币：500
- 初始生命值：20
- 击杀敌人获得金币
- 每波开始额外奖励：50 金币
