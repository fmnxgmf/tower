# Godot 4.6 2D 塔防游戏

## 变更概述

创建一个完整的 2D 塔防游戏，核心特色是开放地图玩家自建路径系统。

## 核心特性

### 游戏机制
- **开放地图建塔**：玩家可在任意位置建造防御塔
- **智能寻路**：A* 算法，建塔时自动验证路径可达性
- **飞行系统**：飞行单位可穿过防御塔但仍遵循寻路
- **资源管理**：初始金币 + 击杀奖励
- **关卡制**：通过 10 波敌人进入下一关

### 防御塔 (5种)
1. 基础炮塔 - 单体伤害
2. 减速塔 - 降低敌人速度
3. 范围塔 - AOE 伤害
4. 狙击塔 - 高伤害长射程
5. 辅助塔 - 增强周围塔（可选）

### 敌人类型 (4种)
1. 快速单位 - 速度快血量低
2. 坦克单位 - 血量高速度慢
3. 飞行单位 - 可飞过防御塔
4. 再生单位 - 持续回血

## 技术实现

### 架构
- Godot 4.6 + GDScript
- 单例模式管理游戏状态
- 场景组件化设计
- 信号驱动的事件系统

### 核心系统
- A* 寻路算法
- 网格地图系统 (20x15)
- 对象池优化
- Area2D 范围检测

### 占位符美术
- ColorRect 表示塔和敌人
- 不同颜色区分类型
- 简单几何形状

## 开发阶段

1. **Phase 1**: 项目初始化和基础框架
2. **Phase 2**: 地图和寻路系统
3. **Phase 3**: 敌人系统
4. **Phase 4**: 防御塔系统
5. **Phase 5**: 战斗系统
6. **Phase 6**: 游戏流程
7. **Phase 7**: 优化和测试

## 预期成果

一个可玩的塔防游戏原型，包含：
- 完整的游戏循环
- 5 种防御塔
- 4 种敌人类型
- 至少 1 个完整关卡（10波）
- 基础 UI 和反馈
